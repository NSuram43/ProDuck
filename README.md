# Pro Duck MVP

React + TypeScript app built with Vite and Tailwind CSS.

## Scripts

- `npm run dev` тАУ start dev server
- `npm run build` тАУ type-check and build
- `npm run preview` тАУ preview production build

## Structure

- `src/components` тАУ reusable components
- `src/pages` тАУ route pages and layout
- `src/utils` тАУ utilities like scoring
- `src/data` тАУ mock data
- `src/hooks` тАУ custom hooks

## Tailwind

Configured via `tailwind.config.ts` and `postcss.config.js`. Styles in `src/index.css`.
